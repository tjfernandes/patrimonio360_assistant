# Queries para demo AMALIA

Fluxos diretos para testar no frontend e com backend a correr.

## 1. Traje

Museu: `mnt`

### Texto

1. Perguntar:

```text
Como se vestiam as crianças no século XVIII?
```

Depois:
- esperar pelos resultados
- clicar no resultado com ponto de interesse
- abrir/ver na tour

2. Perguntar:

```text
Consegues encontrar chapéus de mulher?
```

3. Perguntar:

```text
Mostra-me alguns vestidos com tons mais escuros nesta visita
```

### Imagens

Usar imagens em:

```text
assets/test/mnt/Imagens/
```

Exemplos:

```text
assets/test/mnt/Imagens/MNT0077-3.jpg
assets/test/mnt/Imagens/MNT4703.jpg
assets/test/mnt/Imagens/MNT7619-3.jpg
```

Perguntas para upload:

```text
Consegues identificar esta peça?
```

```text
Que tipo de objeto é este?
```

```text
Mostra-me peças parecidas nesta visita.
```

### Modelos 3D

Usar modelos em:

```text
assets/test/mnt/Modelos3D/
```

Exemplos:

```text
assets/test/mnt/Modelos3D/MNT_28519_vestido_crianca_folhos.glb
assets/test/mnt/Modelos3D/MNT_38746_vestido_branco.glb
assets/test/mnt/Modelos3D/MNT_00967_Bolsa.glb
```

Perguntas para upload:

```text
Em que situações era usado este vestido?
```

```text
Que peça é esta e quais os materiais usados?
```

## 2. Azulejo

Museu: `mnaz`

### Texto

Perguntar:

```text
Encontra o grande panorama de Lisboa.
```

Depois:
- esperar pelo resultado
- navegar para o ponto indicado
- olhar para o panorama na tour

### Imagem

Upload:

```text
assets/test/mnaz/mosteiro_no_painel.jpg
```

Perguntar:

```text
Consegues localizar o mosteiro no painel?
```

Depois:
- abrir o resultado na visita
- clicar na imagem/ponto indicado
- navegar para o mosteiro no painel
